Skin AI – Smart Skin Type Detection & Recommendation System

AI in skin care is an intelligent web application that analyzes a user's skin type and provides personalized skincare guidance. The system uses image analysis and predefined logic to detect skin patterns and then recommends suitable skincare routines, natural remedies, exercises, and product suggestions.

Features

- Skin type detection
- Personalized skincare advice
- Daily skincare routine suggestions
- Recommended skincare products
- Natural home remedies
- Yoga and exercise suggestions for healthier skin
- Weekly feedback system to track improvement
- Progress tracking through skin data

Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- Machine Learning (basic pattern detection)

Project Structure

- "templates/" – Web page templates
- "static/" – Images, CSS, videos, and other static files
- "dataset/" – Skin data used for detection
- "app.py" – Main application file
- "skin_detect.py" – Skin analysis logic
- "create_db.py" – Database creation

Purpose

The goal of this project is to help users understand their skin condition and follow better skincare practices using a simple AI-assisted system.

Future Improvements

- More accurate machine learning model
- Dermatologist-level product recommendation system.
